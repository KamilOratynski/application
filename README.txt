Application displays "x" character as many times as defined by given first parameter.
Character can be changed by setting second parameter.




Aplikacja wyświetla znak „x” tyle razy, ile określa podany pierwszy parametr.
Znak można zmienić ustawiając drugi parametr.
Po wyświetleniu ciągu znaków można zmienić znak określając jego pozycję i nowy znak.
Zmiana jest możliwa do momentu wyjścia z programu.
