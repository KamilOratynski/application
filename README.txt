Aplikacja wyświetla litery zadaną ilość razy oraz umożliwia zmianę pojedynczej literki.

Ilość liter definiujemy poprzez pierwszy parametr.
Domyślna litera to "x", można ją zmienić podając drugi parametr.
Zatwierdzenie "q" spowoduje wyjście z programu.


----


Application displays "x" character as many times as defined by given first parameter.
Character can be changed by setting second parameter.
